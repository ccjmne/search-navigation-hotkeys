"What's this?" bubbles

These bubbles appear when you mouse over an indicator in the results.

Secondary open key

This key can be used as an alternative to the Enter key to:

  • open the focused link
  • toggle the highlighted option in the filter and sort menus

Exit current mode key

This key is used to:

  • close the help card
  • exit filter and sort mode
  • exit switch tabs mode

Secondary navigation keys

These keys can be used as an alternative to the arrow keys when:

  • navigating through the search results
  • navigating through the filter and sort menus
